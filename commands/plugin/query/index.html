Query

Help

Query dependencies in a project

Usage:
  ply query [flags]

Flags:
  -a, --artifactId string   artifactId
  -g, --groupId string      groupId
  -h, --help                help for query
      --target string       Optional target directory (default ".")

Global Flags:
      --debug   turn on debug output
      --json    turn on json output logging

Example usage

plybuild query
